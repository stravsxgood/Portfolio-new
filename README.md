# Web Portfolio
Junior Web Developer berfokus pada pengembangan web yang clean, responsive, dan mudah dikembangkan.

## 🚀 Tech Stack
- **Frontend:** HTML

## ✨ Fitur Utama Project Ini
- **Responsive Design:** Optimal diakses dari perangkat mobile maupun desktop.
- **Dynamic Portfolio:** Menampilkan list project, skills, dan sertifikat secara dinamis.
- **Contact Form / API Integration:** Form kontak terintegrasi untuk menerima pesan dari pengunjung.
- **Clean UI:** Tampilan sederhana, modern, dan mudah dinavigasi.
- **Project Showcase:** Menampilkan detail project, teknologi yang digunakan, dan link demo/repository.
